# SSRVPN Android 安装与权限

导入订阅、连接、状态判断和订阅刷新请先阅读
[公共用户指南](../docs/USER_GUIDE.zh-CN.md)。本页只说明 Android 差异。

## 支持范围

- 当前 APK 仅支持 `arm64-v8a` 设备。
- Android 通过系统 VPN 服务接管 IPv4/IPv6 流量，不使用桌面系统代理模式。

## 安装

1. 从项目正式 Release 或官网固定下载地址获取 `SSRVPN.apk`。
2. 按发布页提供的 SHA256 校验文件核对 APK。
3. Android 提示时，只为本次可信下载来源允许安装未知应用，然后完成安装。
4. 后续版本必须使用同一正式发布证书，才能直接覆盖安装升级。

不要安装聊天群或网盘中无法验证来源、签名和 SHA256 的 APK。

## 首次连接权限

首次点击连接时，Android 会显示系统 VPN 授权窗口。确认后，SSRVPN 通过前台 VPN 服务
维持连接；连接期间通知栏会显示状态和当前节点。

- 拒绝系统 VPN 授权时不会建立连接。
- 同一时间只能有一个系统 VPN；连接前应断开其他 VPN 应用。
- 不要在连接期间强制停止 SSRVPN，否则系统会终止 VPN。
- 可以把 SSRVPN 添加到快速设置磁贴；磁贴连接会沿用最近一次选择的节点。

## 更新与数据

应用内更新下载正式 APK 并校验文件与发布身份。若覆盖安装提示签名不一致，应先确认
现有应用是否来自正式渠道；卸载会清除本地订阅和设置，不要在未记录数据前直接卸载。
